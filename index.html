Centers for Medicare & Medicaid Services — NPPES Provider Tax-ID Verification Service · ADR-003 / ADR-004

Provider Tax-ID Verify

Confirm a provider's Tax ID against the system of record — without ever disclosing it
Verify API Reference

Verification Request

Submit an NPI and a candidate Tax ID. The service returns a boolean only.
POST /api/verify
Verify, don't disclose. The candidate Tax ID is checked inside Oracle via a parameterized EXISTS on (npi, tax_id). No stored Tax ID is ever returned, indexed, or logged — the response carries the boolean only.

Verification Result

Boolean outcome with provider entity type and data currency.
Submit a verification request to see the result.
