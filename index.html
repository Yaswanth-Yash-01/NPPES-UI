National Plan & Provider Enumeration System (NPPES)

Provider Tax-ID Verification Service

Verify API Reference NPI Registry Help
Authorized use only. This system contains U.S. Government information. Verification requests and results are monitored and audited. The candidate Tax ID you submit is checked but never stored, returned, or logged.

Verification Request

Submit an NPI and a candidate Tax ID. The service returns a yes/no result only.
POST /api/verify
Verify, don't disclose. The candidate is checked inside the system of record via a parameterized EXISTS on (npi, tax_id). The stored Tax ID is never returned, indexed, or logged — only the boolean is sent back.

Verification Result

Boolean outcome with entity type and data-currency date.
Submit a verification request to see the result.
